# ST 聊天记录处理器 v0.10.8

SillyTavern Chat Beautifier - 将 SillyTavern 的聊天记录转换为精美格式，支持编辑、书架管理、世界书编辑、角色卡编辑、预设编辑和 AI 辅助功能。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/LYC619/silly-tavern-explorer)

> 🔒 **隐私安全**：所有数据完全保存在本地浏览器，不会上传到任何服务器。

## 🧭 这是什么

一个面向 SillyTavern 用户的**一体化归档工具**：玩完一张角色卡后，把这次游玩的成果在一个地方处理、留存、再利用——

```
导入聊天记录 → 清理/编辑/分章 → 导出归档(TXT/JSONL)
                    ↓ 按这次进度同步更新
        世界书条目  ·  角色卡信息  ·  预设
```

四大模块（聊天处理 / 世界书 / 角色卡 / 预设）都与 SillyTavern 格式互通，导出的文件可直接导回 ST；并在世界书与预设中内置 AI 辅助，按游玩进度沉淀设定、精修条目。书架与沉浸阅读为辅助功能。

## ✨ 功能特性

### 📥 导入与解析
- 支持导入 SillyTavern 导出的 JSON/JSONL 聊天记录
- 自动识别角色名称和用户名称
- 内置正则规则，自动移除思维链、状态栏等元数据

### ✏️ 消息编辑
- 每条消息右上角铅笔图标，点击直接编辑该楼内容 `v0.10.2 改进`
- 修改说话人名称和角色类型
- 删除不需要的消息
- 删除 / 编辑后可一键「撤销」，误操作无需重新导入

### 🔖 楼层导航与收藏 `v0.9.2 新增`
- **左侧悬浮跳转条**：固定在屏幕侧边，不随滚动消失、不挤占正文
- 输入楼层号直达、上一层 / 下一层
- **收藏楼层**：轻量个人书签（独立于章节标记、不进导出），收藏列表一键跳转，跟随作品保存
- **常驻楼层号**：每条消息左侧显示楼层号，统一采用 0-based 编号（首楼为 0，对齐 SillyTavern）
- **应用内全文搜索**：搜索正文、显示命中数、上/下一个命中跳转与高亮定位

### 🎨 主题风格
- **典雅书籍** - 装饰边框，古典排版
- **小说排版** - 经典引号，文学风格
- **社交气泡** - 现代聊天界面
- **极简主义** - 清爽干净

### 📑 章节标记
- 单条消息点击添加章节标记
- 批量导入章节（支持【存档节点】格式）
- 自动解析卷名、章节编号、概要和事件
- 章节编号自动转换为"第一章"格式

### 📤 导出选项
- 导出弹窗：统一配置导出范围和清理选项
- 导出为 JSONL（SillyTavern 兼容格式，可直接导入继续游玩）
- 导出为 TXT 纯文本（适配 Markdown 处理器）
- 正则清理后导出，移除思维链等杂项内容
- 自动清空 swipes 冗余数据，文件体积减少 40-50%
- 可选清理插件缓存（LWB 快照等），进一步精简
- 消息范围导出：全部 / 最近N条 / 自定义楼层范围
- 实时导出统计：显示原始大小、预估大小和节省比例
- TXT 导出附归档信息头（标题 / 角色 & 用户 / 导出日期），文件名自动清洗非法字符

### 🔧 正则处理
- 独立侧边栏：编辑规则时可实时预览效果
- 傻瓜式快速添加：输入开始/结束标签自动生成正则
- 内置常用清理规则（思维链、Theatre标签、状态栏等）
- 支持自定义正则规则
- 规则配置自动保存到浏览器
- 可按用户/助手消息分别应用
- **SillyTavern 正则脚本互通** `v0.10.0 新增`：导入 ST 正则脚本 `.json`（支持单脚本 / 数组 / 社区打包格式），导出为 ST 兼容格式；自动转换位置枚举、无损保留 ST 独有字段（trimStrings / substituteRegex 等）

### 📓 总结（分卷 / 日记 / DIY） `v0.11.0 新增`

通过顶部导航栏的「总结」入口进入。把玩过的剧情用 AI 沉淀成可归档的总结，永久留存、随时重生成。三种呈现共用同一套「选楼层 → 挂设定 → 生成 → 编辑保存」引擎。

- **分卷总结**：手动选楼层生成一卷；**跨卷连贯**——生成第 N 卷时自动带上已有前几卷作为「前情」，剧情线与角色状态不断裂。卷号自动递增、起点楼层建议接上一卷
- **角色日记**：以角色第一人称、多篇日记体回顾故事
- **DIY 创作**：自由提示词（文言列传 / 书信体 / 新闻报道等）
- **对齐 SillyTavern 组装**：可选挂载已保存的**预设**（决定上下文顺序）与**世界书**（默认仅注入常驻/蓝灯条目，也可全部或手动勾选），实时显示 token 估算
- **可编辑可微调可重生成**：结果生成后自动暂存、可直接改、可 AI 微调（润色/精简/扩写）；「保存」转永久；一键**重新生成**回填原设置；复制或导出 `.md`
- **模板库互通**：内置三套模板可查看/编辑/另存；与「AI 工具」页自定义提示词互通，便于反复调优
- **持久化**：生成自动暂存（保留最近若干份）、「保存」永久留存、纳入完整数据备份

### 🪪 角色卡编辑 `v0.10.4 升级`

通过顶部导航栏的「角色卡」入口进入。导入 SillyTavern 角色卡，编辑核心字段后导出归档。

- **支持 PNG 与 JSON**，自动识别 V1 / V2 / V3（同时含 V2/V3 数据时优先 V3）
- **可编辑核心字段**：名称 / 昵称 / 作者 / 版本 / 标签 / 描述 / 性格 / 场景 / 开场白 / 备选开场白 / 对话示例 / 系统提示 / 历史后指令 / 作者备注（标签与备选开场白可增删）
- **导出 PNG（回写）**：从 PNG 导入的卡，编辑后导出 PNG——数据写回图片 tEXt 数据块、**保留原立绘**、可直接导回 SillyTavern，中文不乱码；从 JSON 导入的卡仅可导出 JSON
- **无损保留**：只覆盖编辑过的字段，内嵌世界书、扩展、立绘资源、未识别字段等导出时原样保留
- **内嵌世界书展示**：角色卡内附带的世界书条目一并解析展示（关键词、注入位置、常驻/禁用状态、内容预览）；只读，需编辑请用独立的「世界书」编辑器
- **持久化**：导入自动保留最近 5 份历史、切页不丢编辑、「保存」永久留存（在「已存角色卡」中管理）、纳入完整数据备份

### 🎚️ 预设可视化编辑 `v0.10.3 新增`

通过顶部导航栏的「预设」入口进入。导入 SillyTavern 导出的 Chat Completion 预设 `.json`，可视化查看与编辑，导出可直接导回 ST。

- **概览**：API 来源 / 模型 / 最大上下文 / 采样参数（temperature、top_p、top_k…）/ 提示词块与激活条目统计
- **提示词编辑**：拖拽重排激活顺序、启用 / 禁用、移除（带撤销）、展开编辑内容；未激活条目进「提示词库」可一键加入，自动标记「未引用 / 空条目」
- **新建块 + AI 改写** `v0.10.7 新增`：可手动「新建块」加入自定义提示词；展开编辑某块时用「AI 改写」按要求（让指令更清晰 / 精简 / 强调中文 / 加强沉浸…）改写该块内容并直接替换（保留 `{{char}}`/`{{user}}` 宏，可撤销）
- **实时预览**：按激活顺序渲染启用条目，替换 `{{char}}` / `{{user}}`，估算 token，角色色条区分 system / user / assistant / 系统插槽
- **多角色组**：预设含多个 `prompt_order` 分组时可切换；撤销 / 重做覆盖激活顺序调整
- **工具字段**：编辑 `impersonation_prompt` / `wi_format` / `scenario_format` 等工具型字段
- **内嵌正则**：编辑预设的 `extensions.regex_scripts`，含实时测试区，ST 独有字段无损保留
- **导出**：完整 / 智能 / Markdown 三种导出；保留全部未识别字段无损 round-trip；附「变更对比」
- **持久化**：导入自动保留最近 5 份历史、切页不丢编辑、「保存」永久留存（在「已存预设」中管理）、纳入完整数据备份

### 🌍 世界书可视化编辑器 `v0.8 新增`

完整的 SillyTavern 世界书（World Info / Lorebook）可视化编辑工具，通过顶部导航栏的「世界书」入口进入。

- **导入/导出**：支持导入 SillyTavern 世界书 JSON（兼容记录格式与数组格式），导出为标准 JSON
- **格式加固** `v0.10.0`：正确处理 ST 的 `disable` 字段（修复禁用条目被读反的 bug）、导出时强制条目记录键与 uid 一致（修复导出后无法重新导入）、兼容老格式（字符串位置 / snake_case 旧字段名 / 废弃取值）
- **追加导入**：将新世界书追加到已有数据，自动处理 UID 冲突
- **卡片 & 列表视图**：双视图切换展示条目，卡片显示策略标识（🔵常驻/🟢关键词/🔗向量）、关键词标签及基于 Inclusion Group 的 HSL 颜色条
- **完整编辑面板**：支持所有 SillyTavern 世界书字段——策略切换、逻辑模式、递归设置、深度与角色配置；内容字段可「放大」弹窗全屏编辑
- **置顶工具栏 + 分页** `v0.10.1`：搜索 / 排序 / 筛选 / 每页条数 / 视图切换两行置顶不随列表滚动；筛选项收进「筛选」弹窗（带激活数角标）；分页（25/50/100/全部）应对大体量世界书（数百上千条）
- **搜索筛选排序**：按标题/关键词/内容搜索，按策略和启用状态多维筛选，按 Order/标题/UID 排序
- **批量操作**：批量改位置/策略/启用状态、批量加前缀、批量删除；支持 **Shift 连选**（点开头 + Shift 点结尾选整段）
- **AI 追加条目** `v0.10.5 新增`：顶部「AI 追加」按钮，根据当前正在编辑的聊天记录，让 AI 提炼**新增**的世界观设定追加进世界书——喂入已有条目防重复、可选楼层范围、确认后追加（自动避免 UID 冲突），与「AI 工具」共享 API 配置
- **单条 AI 改写** `v0.10.6 新增`：编辑某一条目时，「内容」区/「放大编辑」窗口的「AI 改写」可按要求（优化措辞/精简/扩写/改人称…）改写**这一条**内容并直接替换，可撤销
- **单条直接删除 / 关键词批量编辑 / 搜索仅标题** `v0.10.5`：条目可直接删除（带撤销）无需进批量模式；关键词支持「批量编辑」文本框整体增删改；搜索可切换「搜全部 / 仅标题」
- **快速创作模式**：粘贴空行分隔的世界观设定文本，自动拆分为条目、提取标题和关键词
- **前缀归类**：扫描已有标签（group 字段），为未分类条目批量分配标签并自动添加「标签--标题」前缀、重新编号
- **自动历史 + 持久化** `v0.10.1`：导入自动保留最近 5 份历史、切换页面不丢失编辑；点「保存到书架」永久留存
- **移动端适配**：底部抽屉式编辑器，触控友好

### 📚 书架功能
- 将处理后的聊天记录保存到本地书架
- 支持自定义封面图片，无封面时自动生成渐变占位封面
- 支持按更新时间、创建时间、标题排序
- 作品管理：编辑标题、删除作品
- 使用 IndexedDB 存储，支持大量数据

### 📖 沉浸式阅读
- 点击书架作品可选择「沉浸阅读」或「编辑处理」
- 小说风格阅读界面，纸张质感背景
- 手机端滑动翻页，桌面端点击/键盘翻页
- 可调节字体大小，支持多种字体（宋体/楷体/黑体/等宽）
- 阅读进度条，支持日间/夜间模式

### 🌙 界面与体验
- 日间/夜间模式切换
- 导航时自动保存工作状态（不会丢失进度）；大记录的正文本体存于 IndexedDB，跨页临时态只存轻量指针，彻底告别大记录切页丢失 `v0.10.2 改进`
- 设置自动持久化到浏览器
- 「清除临时缓存」一键自救：切页后内容异常时清空页面间临时态，不影响书架与世界书 `v0.10.2 新增`
- 响应式设计，完美适配手机/平板/桌面
- 新手引导：聊天处理 / 书架 / 世界书 / 角色卡 / 预设 / AI 工具各模块首次进入自动分步引导，可在设置中重置
- **长记录流畅**：阅读区采用虚拟化渲染，数百楼/数十万字的记录也能秒开、滚动顺滑
- **完整数据备份**：书架页一键备份/恢复全部作品、世界书、角色卡**与预设**（导出为 JSON 文件）

### 🤖 AI 工具箱
- **总结剧情**：分析聊天内容，生成结构化剧情概要，并可自动生成章节标记
- **提取世界书**：从对话中提取世界观设定，生成世界书条目导入「世界书」编辑器
- **平行世界**：基于现有剧情生成分支续写提纲
- **自定义提示词**：自由编写提示词处理选定楼层；所有内置模板的提示词均可自行编辑
- **批量处理**：按段落并行处理长对话，加速分析
- 配置一次，全局共享：API Key / 接口地址 / 模型在「世界书」「预设」的 AI 功能间通用
- 支持 OpenAI 官方 API、各类兼容中转站、本地兼容模型（如 Ollama）；API 密钥仅保存在本地浏览器

> 除独立的「AI 工具」页外，**世界书**支持「AI 追加条目 / 单条 AI 改写」，**预设**支持「提示词块 AI 改写」——详见上文各模块说明。

## 🚀 快速开始

### 在线使用

访问部署好的网站即可直接使用。

### 本地一键启动

本项目提供一键启动脚本，无需复杂配置：

#### Windows 用户
1. 确保已安装 [Node.js](https://nodejs.org/)（推荐 v18+）
2. 双击 `start.bat`
3. 浏览器会自动打开 `http://localhost:4173`

#### Mac/Linux 用户
1. 确保已安装 [Node.js](https://nodejs.org/)（推荐 v18+）
2. 打开终端，进入项目目录
3. 运行 `npm install && npm run build && npm run preview`
4. 浏览器访问 `http://localhost:4173`

### 使用流程

**核心：处理并归档聊天记录**
1. 导入 SillyTavern 聊天记录（JSON / JSONL）
2. （可选）点击每楼右上角铅笔修改消息、调整主题、添加章节标记
3. （可选）配置正则规则清理思维链 / 状态栏等杂项
4. （可选）保存到书架
5. 导出为 JSONL（可导回 ST 继续玩）或 TXT（归档阅读）

**一体化归档：按这次游玩进度更新其他资产**
- 进入「世界书」：编辑世界观条目，或用「AI 追加」按本次聊天提炼新设定、「AI 改写」精修某条
- 进入「角色卡」：导入角色卡，按剧情进度更新描述等字段，导出 PNG（保留立绘）/ JSON 归档
- 进入「预设」：查看 / 编辑本次用的预设，「新建块」或「AI 改写」微调提示词
- 在书架页一键备份 / 恢复全部作品、世界书、角色卡、预设

## 📝 批量导入章节格式

支持以下格式的章节总结文本：

```markdown
### 存档节点：第一卷 - 初遇

#### 【本卷概要】

描述本卷的主要剧情...

#### 【关键事件索引】

- **初次相遇**: 描述事件的起因、经过和结果...
- **命运转折**: 另一个重要事件的描述...
```

## 🤖 AI 工具使用说明

1. 进入"AI 工具"页面
2. 配置 API Key（支持 OpenAI 及兼容接口）
3. （可选）展开高级设置，自定义接口地址和模型
4. 选择需要的工具，粘贴内容后点击生成
5. 复制结果到相应位置使用

**支持的 API 格式：**
- OpenAI 官方 API
- 各类 OpenAI 兼容中转站
- 本地部署的兼容模型（如 Ollama + OpenAI 兼容层）
